Progress Meters

Progress meters can be set for TeamRaiser events, TeamRaiser participants, TeamRaiser teams or TeamRaiser local companies. The frId attribute must be set unless passing object data using the progressData attribute. For the meter to show up, a goal must be set. For example, if setting a company meter and no goal is set on the local company it will not show.

Examples

Event with showMeterPercent set. Meter will only show when 5% or more.

<div progress-meter type="event" show-meter-percent="5" fr-id="1070"></div>

Setting type as individual passing constituent id.

<div progress-meter type="individual" id="1001491" fr-id="1070"></div>

Passing progress data as object. Must be passed with amount and goal properties. This is the preferred method when data is already available on page.

<div progress-meter progress-data="{ amount: ctrl.amount, goal: ctrl.goal }"></div>

Setting type as team and passing team id as object.

<div progress-meter type="team" fr-id="1070" id="{{ctrl.teamId}}"></div>
