Donation Levels

Provides a button list of donation levels either pulled from the TeamRaiser Configuration for additional donation during registration (must set frId) or from a donation form (must pass formId).

Examples

Pull in list of donation levels from a donation form with large buttons.

<div donation-levels form-id="1400" button-size="lg"></div>

Pull in list of donation levels using donationLevels object set in parent controller. Example can be seen in this pages Controller.

<div donation-levels donation-levels-data="donationLevels"></div>
