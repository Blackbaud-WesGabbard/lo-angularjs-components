Participation Types

Provide a list of active participation types for a TeamRaiser Event. Must either set frId or pass object using participantListData attribute. Defaults to horizontal layout, but can specify vertical layout. For horizontal layout, if fee or minimum goal is greater than zero it will be included in an unordered list. Participation type description will be included as last item in unordered list. Vertical layout used for registration and will make the particpation type selectable.

Examples

Horizontal List of Participation Types and passing participationTypes object to populate. Example can be seen in this pages Controller

<div participation-types participation-types-data="participationTypes"></div>
