Luminate Online AngularJS Component Library

Progress Meters

Properties

Attribute Format Required Description
type String Required Type of meter. Can be set as either Event, Individual, or Team
frId String or Object Optional if embedded in TeamRaiser, otherwise required. The frId or TeamRaiser id.
id String or Object Required for Individual or Team. Pass Constituent ID for Individual, or Team ID for Team. Can be passed as string or as an object.
data Object Optional Can pass the meter data as object if data already present. Must pass amount and goal
showMeterPercent String Optional Can set the minimum percent of the meter before it will show

Example

Event

<div progress-meter type="event" show-meter-percent="5" fr-id="1070"></div>

Individual

<div progress-meter type="individual" id="{{ctrl.consId}}"></div>
