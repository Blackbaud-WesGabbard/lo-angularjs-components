Progress Meters

Progress meters can be set for TeamRaiser events, TeamRaiser participants, TeamRaiser teams or TeamRaiser local companies. The frId attribute must be set if using outside of a TeamRaiser. For the meter to show up, a goal must be set. For example, if setting a company meter and no goal is set on hte local company it will not show.

Attribute Format Description
type String Required, can be set as Event, Individual, Team or Company.
frId String or Object TeamRaiser id is optional if embedded in TeamRaiser, otherwise required.
progressData Object Pass the progressData as an object instead of id if data already present. Must pass amount and goal properties. This is preffered method when data aleady in scope to prevent redundant API calls.
id String or Object If not passing progressData object, then must id attribute. Pass Constituent ID for Individual, Team ID for Team or Company ID for Company. Can be passed as string or as an object. Not needed for Event Meters.
showMeterPercent String Can set the minimum percent of the meter before it will show

Examples

Event with showMeterPercent set. Meter will only show when 5% or more.

<div progress-meter type="event" show-meter-percent="5" fr-id="1070"></div>

Individual setting constituent id as String.

<div progress-meter type="individual" id="1001491"></div>

Individual passing progress data as Object. This is the preferred method when data is already available on page.

<div progress-meter type="individual" progress-data="{ amount: ctrl.amount, goal: ctrl.goal }"></div>

Team setting id as team id Object

<div progress-meter type="team" id="ctrl.teamId"></div>

Company setting id as String

<div progress-meter type="company" id="1013"></div>

Participation Types

Provide a list of active participation types for a TeamRaiser Event. Defaults to horizontal layout, but can specify vertical layout. Vertical layout used for registration and will make the particpation type selectable. If fee or minimum goal is greater than zero, will be included in an unordered list. participation type description will be inlcuded as last item in unordered list.

Examples

List of Participation Types

<div participation-types></div>

Vertical List of Participation Types

<div participation-types layout="vertical"></div>
